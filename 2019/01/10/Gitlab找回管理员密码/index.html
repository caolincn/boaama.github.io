Gitlab找回管理员密码

发表于 2019-01-10 | 分类于 Git

真的是悲催，在需要添加Gitlab用户的时候发现管理员密码忘记了，而且没有邮箱服务器，因此不能用邮箱找回功能。由于第一次部署Gitlab服务器，经验不足，后续可能会遇到的问题都还没有经历过，现在第一个问题到来了，我该用什么办法来解决呢？难道要重建？
重建当然能解决问题，但肯定不是一个好的办法，后面我了解到Gitlab是有留下修改管理员账号密码方法的。

尝试输入密码无数次无果后，开始了管理员账号密码找回的工作。

下面为找回的步骤：
如果是以Docker方式启动的，首先执行 docker exec -it <容器id> bash 进入容器中。

  1. 执行 gitlab-rails console 登陆Gitlab的rails控制台
  2. user = User.where(username: 'root').first 定位到id=1的用户，就是超级管理员
  3. user.password = 'password' 设置新的密码
  4. user.save! 保存设置

至此密码找回工作完成。

让我们再次登陆，验证此方法是否成功重置了管理员密码

登陆成功！！！
